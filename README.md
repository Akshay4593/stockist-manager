**STOCKLIST**
- IOS application written using Swift 6. It uses MVVM architecture.

**Prerequisites**
- Xcode 16.0 Swift 6 Deployment Target 15.0

**FEATURES**
- MVVM Architecture
- Offline Support
- Custom UI Components
- No Third-Party Dependencies
- Customizable and Scalable
